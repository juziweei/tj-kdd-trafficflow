# V8 增强特征说明

## 新增特征（10个）

### 1. 时间模式特征（3个）
- `slot_mean`: 同时段历史均值（如每天8:00的平均流量）
- `slot_std`: 同时段历史标准差
- `slot_count`: 同时段历史样本数

### 2. 趋势特征（2个）
- `short_trend`: 短期趋势 = (lag1 - lag72) / 72
- `long_trend`: 长期趋势 = (lag72 - lag504) / 432

### 3. 波动性特征（2个）
- `volatility`: 最近6个窗口的标准差
- `cv`: 变异系数 = std / mean

### 4. 天气交互特征（3个）
- `temp_morning`: 温度×早晨标志
- `rain_rush`: 降水×高峰期标志
- `wind_temp`: 风速×温度

## 预期提升

- 时间模式: +0.3~0.5 MAPE
- 趋势特征: +0.2~0.4 MAPE
- 波动性: +0.1~0.3 MAPE
- 天气交互: +0.2~0.4 MAPE

**总预期: 0.8~1.6 MAPE 改进**
