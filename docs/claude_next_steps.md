# Claude的下一步优化建议（基于14.75 MAPE）

## 当前状态对比

**Claude路线**: 14.75 MAPE（合并数据）
**Codex路线**: 16.72~17.82 MAPE（TFT+Optuna进行中）

## 我的3个优化方向

### 1. 时间窗口扩展（预期-0.3~0.5）
当前只用了2个月数据，可以：
- 扩展到phase2的10月数据
- 增加季节性特征

### 2. 集成学习（预期-0.2~0.4）
- 训练3个独立模型（不同随机种子）
- 简单平均或加权融合
- 减少单模型方差

### 3. 后处理优化（预期-0.1~0.3）
- 异常值平滑（MAPE对异常敏感）
- 物理约束（流量非负、连续性）
- 分位数回归（处理极端值）

## 与Codex的协同

- Codex: TFT+Optuna（复杂架构）
- Claude: 数据+集成（稳健提升）

目标：**14.0以下**
